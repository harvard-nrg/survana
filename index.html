Mobile

Survana works on any device that supports HTML5 and Javascript.

Secure

All data are encrypted using industry-standard AES256 and state of the art elliptic curve cryptography.

Open Source

Survana is licensed under the New BSD License. The code is available on GitHub.

Scalable

Survana's module architecture allows for numerous deployment and scalability scenarios.

Works Offline

Network failures do not impact the progress of surveys. We encrypt and save data locally, until the network connection becomes available.

Theme Support

Survana supports theming of questionnaires, with user-supplied themes and files coming soon.
